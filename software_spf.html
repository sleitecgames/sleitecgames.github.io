Hazelnut is a 2D Mobile Game (Android & iOS)
The player must try to catch all the dropping hazelnuts
which are dropped by the Evil Terrifying Squirrel.
The squirrel also drops evil objects, which shouldn't be catched!

Using:
  • C++11
  • cocos2d-x engine
  • Visual Studio 2013/2015 (Windows), XCode (OS-X / iOS), Android Studio
  • Gimp (Programmer Art)
